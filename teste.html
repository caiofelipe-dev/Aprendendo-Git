Aprendendo Git e GitHub

O que é Git?

Git é um sistema de controle de versão distribuído, usado principalmente para gerenciar o código-fonte de projetos de software. Ele permite que múltiplos desenvolvedores trabalhem no mesmo projeto simultaneamente.

O que é GitHub?

GitHub é uma plataforma de hospedagem de código-fonte que usa o Git como sistema de controle de versão. Ele oferece ferramentas de colaboração, como pull requests, issues e wikis, para facilitar o desenvolvimento de software em equipe.

Comandos básicos do Git

ALTERAÇÃO

  • git init: Inicializa um novo repositório Git.
  • git clone [url]: Clona um repositório remoto para o seu computador.
  • git add [arquivo]: Adiciona um arquivo ao índice (staging area).
  • git commit -m "[mensagem]": Cria um commit com os arquivos adicionados.
  • git push: Envia os commits para o repositório remoto.
  • git pull: Atualiza o repositório local com as mudanças do repositório remoto.
